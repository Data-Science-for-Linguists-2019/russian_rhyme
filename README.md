## Files and directories

### Code

#### Notebook

* [Progress report 1 notebook](dev/progress_report_1.ipynb)

#### Library

* [*cyr2phon.py*](cyr2phon/cyr2phon.py): Cyrillic to phonetic library; exposes `cyr2phon.transliterate()`
